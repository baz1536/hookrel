Reset Your Password

HookRel does not have self-service password reset. To regain access:

1
If you are a regular user — ask an admin to reset your password via Settings → User Management.
2
If you are the admin — you can bypass authentication temporarily to reset your password:
  1. In your Docker environment, set AUTH_ENABLED=false
  2. Restart the container
  3. Log in, go to Settings → User Management → Edit User, and set a new password
  4. Set AUTH_ENABLED=true and restart the container again
